# Uai Odontologia – Site Oficial  

Uma página única, moderna e responsiva que apresenta os serviços odontológicos, a equipe clínica e depoimentos de pacientes da **Uai Odontologia**. O design usa tons de azul, animações de rolagem suave, carrosséis interativos e integração com WhatsApp para agendamento de consultas.  

---  

## 🚀 Recursos  

- **Design Responsivo** – Otimizado para mobile, tablet e desktop com Tailwind CSS.  
- **Carrosséis Interativos**  
  - *Tratamentos*: cards deslizantes (Implantes, Facetas, Clareamento, etc.).  
  - *Equipe*: perfis rotativos de dentistas e especialistas com credenciais.  
- **Animações Suaves** – Elementos aparecem e deslizam ao rolar a página (Intersection Observer).  
- **Cabeçalho Dinâmico** – Transparente no topo e sólido ao rolar.  
- **Integração com Google Maps** – Mapa embutido para as duas unidades da clínica.  
- **Integração com WhatsApp** – Botão flutuante e CTAs com mensagem pré‑preenchida.  
- **Seção de Depoimentos** – Grade com avaliações e classificação por estrelas.  

---  

## 🛠️ Tecnologias Utilizadas  

| Tecnologia | Uso |
|------------|-----|
| **HTML5** | Estrutura semântica. |
| **Tailwind CSS** | Estilização utilitária via CDN. |
| **Swiper.js** | Slider com suporte a toque (tratamentos e equipe). |
| **Iconify** | Ícones (Material Design). |
| **JavaScript puro** | Menu móvel, efeitos de rolagem, inicialização do carrossel. |
| **Google Fonts** | Fonte *Inter*. |
